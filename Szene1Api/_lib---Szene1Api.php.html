File: Szene1Api.php

Source Location: /lib/Szene1Api.php

A minimalist, modern and CURL-free Library for PHP 5.3+ to access the Szene1 http://szene1.at XML REST API

Documentation for the API is available at http://wiki.szene1.co.at/wiki/Weblife1_API This library uses the HTTP fopen wrapper of PHP, make sure it's enabled. You can also subclass Szene1\Api and override the request() Method
